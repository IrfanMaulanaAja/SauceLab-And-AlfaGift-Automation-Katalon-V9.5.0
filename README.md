# SauceLab-And-AlfaGift-Automation-Katalon-V9.5.0
Automation Test Mobile Android Using Katalon Studio 9.5.0

Preparation for running test using katalon studio v9.5.0
- Install Katalon Studio
- Install nodejs
- Install appium version 3.5.0
- Install jdk 21.0.2
- Install "Android.SauceLabs.Mobile.Sample.app.2.7.1.apk"
- Real device mobile android

How to running test :
- clone project form git
- Open project using katalon studio
- Open folder test suites
- Click SauceLabsAndAlfaGift
- Setting Config (Execution Mode : Sequential, Run with : Android, Run Configuration : your device)
- Click Execute

Notes :
- Keep the phone screen on.
